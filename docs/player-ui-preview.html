設計說明 — 一致性決策
🔗 共用 Token（兩個 UI 都遵守）
R
Radius scale 統一：xs=4 · sm=6 · md=10 · lg=16 · xl=22 · full=9999px。Popup 與 Player 的按鈕、卡片、面板都使用同一比例，不再各自為政。
F
Font stack 統一：-apple-system, SF Pro Text, Helvetica Neue, Arial。Player 舊版用了 "Avenir Next"，改為對齊 popup 與 dashboard。
M
Motion 統一：ease-out cubic-bezier(0.16,1,0.3,1)，dur-short=160ms，dur-mid=240ms。兩個 UI 的 transition 感受一致。
A
Accent 橋接：Dashboard 用單色黑白；Player 用 amber #F0BC56 作為主動色。Popup 中的 AI assessments stat 也用 #34C759 green 呼應。
🆕 Popup 主要改動
1
Status bar 新增（Header 下方）：Active / 當前域名 / AI mode — 讓使用者一眼知道現在保護狀態，無需往下滾動。
2
Player chip 重設計：chip 加 status dot，active chip 反色（黑底白字），更清楚表示「鎖定中」vs「待選」。
3
Stats 第 4 欄：新增「AI evals」統計，與 dashboard overview 對齊（4 個 stat card）。
4
AI monitor panel 精簡：移除長篇 evidence 文字，改為 2×2 key-value grid（tier / score / provider / mode），gate detail 按需展開。
🎬 Player v6 主要改動
1
3 折疊分區取代 5 固定 section：播放控制（含 Timeline + Volume + Speed）/ 畫面調整 / 防護設定。防護設定預設收合，省空間。
2
Transport 改為 6 按鈕橫排：-10s · 靜音 · ▶播放 · Loop · +10s · 全螢幕，從 3×2 大卡片改為緊湊行列，視覺更接近 preview 舊版風格。
3
畫面調整新增色調 Tint + 色溫 Temp：分兩子組（光影 Luminance / 色彩 Color），SVG feColorMatrix 實作色溫，CSS hue-rotate 實作色調。
4
移除 AI panel：播放器不再顯示 AI 評估資訊，保持無干擾播放的核心定位。AI 資訊回歸 dashboard 管理。
